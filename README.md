# csci-456-frontend-demo
A static React Native mobile app to demonstrate understanding of the framework.

## Features
- Displays a list of events
- Allows navigation to an event detail screen
- Simulates event registration (static confirmation)
- Demonstrates reusable components and state management

## Tasks
1. Set up React Native project.
2. Create Event List and Event Detail screens.
3. Implement static event data (no backend connection yet).
4. Set up navigation between screens.
5. Simulate event registration (static confirmation, no real data stored).

## Setup and Testing Instructions
### PREREQUISITES: Node.js must be installed on your computer, Expo Go app needed for testing
### 1. Clone the Repository
```
git clone https://github.com/ahrenagatep/csci-456-frontend-demo.git
```
### 2. Open the folder in your IDE
### 3. Install Dependencies
```
npm install
```
### 4. CD into the frontend folder
```
cd frontend
```
### 5. Start Expo
```
npx expo start
```
### 6. Scan the QR code using the Expo Go app!
